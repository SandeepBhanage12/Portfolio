===========================================
EMAILJS ENVIRONMENT VARIABLES SETUP
===========================================

Create a file named ".env" in the vibrant-bio-web folder with the following content:

VITE_EMAILJS_SERVICE_ID="service_nu6hrfp"
VITE_EMAILJS_TEMPLATE_ID=your_template_id_here
VITE_EMAILJS_PUBLIC_KEY=your_public_key_here

Replace the placeholder values with your actual EmailJS credentials.

See EMAILJS_SETUP.md for detailed instructions on how to get these values.
